# NFT Card Games

## Testing
~~~
$ npm i
$ npx hardhat test test/e2e.test.js
~~~

## Hardhat
This project is built on hardhat thus the following may be used:
~~~
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
~~~
